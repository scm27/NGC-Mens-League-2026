How to Publish to the Site
1
Export from Squabbit

After scoring each week, export the round as a CSV file from the Squabbit tournament manager.

2
Preview it here first

Drop the CSV on the left to confirm it parses correctly — you'll see the match points and team scores.

3
Run the update script

Copy the CSV into your repo's data/ folder, then run python build.py to rebuild league_data.json.

4
Commit & push

Run git add . && git commit -m "Add Week N" && git push — GitHub Pages updates in ~30 seconds.
